# Intervallo

---

## is 何

音声のピッチをVSQXなどの他のデータを使って変更するツール

## Requirements

* >= Windows 7
* >= .NET Framework 4.5.2

## How to use

1. ファイルを開きます
2. 音階データを読み込みます
3. 保存します

end
